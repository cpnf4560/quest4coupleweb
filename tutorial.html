📖 O que é o Quest4Couple?

O Quest4Couple é uma ferramenta interativa para casais descobrirem compatibilidades, desejos e fantasias de forma privada, segura e divertida.

💑

Descubram-se juntos!

🎯 Como funciona?

1
Exploração Individual
Cada um responde aos questionários individualmente e em privado, sem ver as respostas do parceiro.
2
Partilha Segura
Trocam um código de acesso ou ficheiros para comparar respostas.
3
Descoberta Mútua
O sistema compara as respostas e revela apenas o que ambos estão interessados!
✅ Privacidade Total

As respostas são encriptadas e só vocês têm acesso. Nem nós conseguimos ver o que respondem!

🔐 Porquê Fazer Registo?

💡 Podes VER sem registo, mas NÃO podes RESPONDER

O registo é 100% gratuito e permite guardar as tuas respostas de forma segura na nuvem.

✨ Vantagens do Registo

☁️
Acesso Multiplataforma
Responde no PC, telemóvel ou tablet - sincronizado automaticamente!
🔗
Partilha Fácil
Gera um código único para o parceiro aceder às tuas respostas.
💾
Backup Automático
Nunca percas as tuas respostas - tudo guardado na cloud!
📊
Relatórios Detalhados
Adiciona o teu parceiro pelo código de acesso e compara respostas diretamente no dashboard.

O teu email e password são protegidos pelo Firebase Authentication do Google. As respostas são guardadas de forma encriptada.

💬 Comentários (Opcional)

Podes adicionar um comentário a cada resposta para explicar melhor o que pensas. Isto ajuda na comparação!

💾 Autosave

Se estiveres autenticado, cada resposta é guardada automaticamente:

⚡
Imediato
Ao clicar em Por favor!/Yup/Meh.../Talvez, a resposta é guardada instantaneamente no Firestore.
🕐
Comentários com Delay
Comentários são guardados 1 segundo após parares de escrever (debounce).
📊
Progresso Atualizado
A barra de progresso e o dashboard são atualizados automaticamente.

💾 Guardar & Carregar

Existem 2 formas de guardar as tuas respostas:

☁️ 1. Guardar na Nuvem (Automático)

✨ Recomendado!

Se estiveres autenticado, as respostas são guardadas automaticamente no Firebase. Não precisas de fazer nada!

💻 2. Guardar Localmente (Ficheiro)

Se preferires, podes exportar as respostas para um ficheiro:

1
Clicar "💾 Guardar Respostas"
No final da página de questionários.
2
Definir Código de Segurança
Insere um código para encriptar o ficheiro. Guarda-o bem!
3
Ficheiro Descarregado
Um ficheiro .q4c é descarregado para o teu computador.

📂 Carregar Respostas de Ficheiro

1
Clicar "📂 Carregar Respostas"
No início da página de questionários.
2
Selecionar Ficheiro .q4c
Escolhe o ficheiro guardado anteriormente.
3
Inserir Código
Digita o mesmo código de segurança usado para guardar.
⚠️ Importante

Guarda o código num local seguro! Sem ele, não consegues desencriptar o ficheiro.

🤝 Sistema de Conexões

O sistema de conexões permite que tu e o teu parceiro se conectem de forma segura para compararem respostas. Agora com pedidos de conexão que precisam ser aceites!

🔒 Segurança Melhorada

Agora, quando alguém envia um pedido de conexão, tu precisas aceitar antes de poderem ver as respostas um do outro. Isto garante que ninguém pode aceder às tuas respostas sem autorização!

📤 Enviar Pedido de Conexão

1
Ir ao Dashboard ou Relatório
Acede ao Dashboard ou página de Relatório onde podes gerir conexões.
2
Clicar "Adicionar Parceiro"
Encontra o botão para adicionar um novo parceiro.
3
Inserir Código do Parceiro
Introduz o código de acesso do teu parceiro (ex: ABC-XYZ-123).
4
Pedido Enviado!
O parceiro receberá uma notificação com o pedido pendente.

📥 Receber e Gerir Pedidos

🔔
Notificação de Pedido
Quando recebes um pedido, aparece um badge vermelho no menu e uma secção "Pedidos de Conexão" no Dashboard e Relatório.
✅
Aceitar Pedido
Clica no botão "✅ Aceitar" para estabelecer a conexão. A partir deste momento, podem comparar respostas!
❌
Rejeitar Pedido
Clica no botão "❌ Rejeitar" se não reconheces a pessoa ou não queres conectar.
📨 Pedido de Conexão
De: João Silva
✅ Aceitar ❌ Rejeitar

🗑️ Eliminar Conexão

Se precisares de remover uma conexão existente:

1
Ir às Conexões
No Dashboard ou Relatório, encontra a lista de parceiros conectados.
2
Clicar "🗑️ Remover"
Cada conexão tem um botão para remover.
3
Confirmar Duas Vezes
Por segurança, precisas de confirmar duas vezes antes de eliminar.
💡 Nota Importante

Ao eliminar uma conexão, o outro utilizador deixa de poder ver as tuas respostas. Podem sempre reconectar mais tarde com um novo pedido.

🔔 Novo: Pedidos de Conexão

Agora os pedidos precisam ser aceites antes da conexão ser estabelecida. Vê mais detalhes na secção 🤝 Conexões.

🆚 Código vs Ficheiro

🔑 Código
  • Mais simples
  • Automático
  • Sempre atualizado
  • Pedidos com aceitar/rejeitar
  • Pode eliminar conexões
📁 Ficheiro
  • Funciona offline
  • Controlo total
  • Backup físico
💡 Recomendação

Usa código se ambos têm conta - muito mais fácil e seguro! Usa ficheiro se preferes privacidade total.

🔒 Segurança

Código único e aleatório. Os pedidos precisam ser aceites antes de poder ver respostas. Pode eliminar conexões a qualquer momento!
