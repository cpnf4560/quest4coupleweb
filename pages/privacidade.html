Política de Privacidade

Última atualização: 17 de novembro de 2025

1. Introdução

A privacidade dos nossos utilizadores é absolutamente fundamental. Esta política explica como tratamos (ou melhor, como NÃO tratamos) os vossos dados.

2. Dados que Recolhemos

Recolhemos o mínimo absoluto necessário para o serviço funcionar:

  • Email: Para criar conta e recuperar password
  • Respostas aos questionários: Armazenadas localmente no vosso navegador
  • Dados de navegação básicos: Para melhorar a experiência do site

3. Como Usamos os Dados

Os vossos dados são usados APENAS para:

  • Permitir que façam login
  • Guardar as vossas respostas (no vosso dispositivo)
  • Mostrar os resultados do matching
  • Melhorar o serviço (análise anónima agregada)

4. O Que NÃO Fazemos

NUNCA fazemos isto:

  • Vender ou partilhar os vossos dados com terceiros
  • Usar os vossos dados para publicidade direcionada
  • Ler as vossas respostas individuais
  • Associar respostas a perfis identificáveis
  • Partilhar informação sem consentimento explícito

5. Armazenamento de Dados

As vossas respostas aos questionários são armazenadas localmente no vosso navegador usando localStorage. Isto significa que:

  • Nós não temos acesso direto às vossas respostas
  • Os dados ficam no vosso dispositivo
  • Se limparem o histórico do navegador, as respostas são apagadas

6. Segurança

Usamos encriptação HTTPS para todas as comunicações. As passwords são encriptadas e nunca são armazenadas em texto simples.

7. Os Vossos Direitos (RGPD)

De acordo com o RGPD, têm direito a:

  • Aceder aos vossos dados
  • Corrigir dados incorretos
  • Eliminar a vossa conta e todos os dados
  • Exportar os vossos dados
  • Opor-se ao processamento dos dados

8. Cookies

Usamos cookies essenciais para o funcionamento do site (ex: manter sessão ativa). Não usamos cookies de tracking ou publicidade.

9. Alterações a Esta Política

Podemos atualizar esta política ocasionalmente. Notificaremos sempre sobre mudanças significativas através do email registado.
