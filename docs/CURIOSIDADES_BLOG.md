## 🔧 Como Funciona

1. Quando um artigo é carregado em `artigo.html`, o sistema chama `loadAndInsertCuriosities(articleId)`
2. Essa função busca dados de `publicStatistics/questionAnalytics` no Firebase
3. Para cada curiosidade mapeada, procura a questão correspondente nos dados de analytics
4. Calcula as taxas de aceitação usando a fórmula: `(porfavor + yup + talvez×0.5) / total`
5. Substitui os placeholders `{...}` pelos valores reais
6. Insere os blocos HTML no conteúdo do artigo, após headings `<h3>` ou `<h4>`
